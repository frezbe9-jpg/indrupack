# Индивидуальные Решения Упаковки (ИРУ)

## 📦 О проекте

Современная B2B-платформа для компании, специализирующейся на производстве гофроупаковки и картонных коробок. Система включает клиентский интерфейс, личный кабинет для отслеживания заказов и защищенную админ-панель для управления заявками.

### 🎯 Ключевые особенности

- **Интерактивный квиз** для подбора оптимальной упаковки
- **Каталог стандартов FEFCO** с техническими характеристиками
- **Личный кабинет клиента** с отслеживанием статуса заказов в реальном времени
- **Защищенная админ-панель** с аналитикой и управлением заявками
- **Telegram-интеграция** для мгновенных уведомлений о новых заказах
- **Адаптивный дизайн** с темной темой и современным UI/UX

## 🚀 Быстрый старт

### 1. Клонирование и установка

```bash
# Клонирование репозитория
git clone <repository-url>
cd iru-packaging

# Установка зависимостей
npm install
```

### 2. Настройка окружения

Создайте файл `.env` на основе `.env.example`:

```env
# База данных
DATABASE_URL="postgresql://user:password@localhost:5432/iru_db"

# Безопасность
ADMIN_SECRET="ваш_секретный_код_для_админки"

# Telegram (опционально)
TELEGRAM_BOT_TOKEN="your_bot_token"
TELEGRAM_CHAT_ID="your_chat_id"

# Приложение
NEXT_PUBLIC_APP_URL="http://localhost:3000"
```

### 3. Настройка базы данных

```bash
# Создание базы данных PostgreSQL
createdb iru_db

# Применение миграций
npx drizzle-kit push
```

### 4. Запуск проекта

```bash
# Режим разработки
npm run dev

# Сборка для продакшена
npm run build
npm start
```

Проект будет доступен по адресу: `http://localhost:3000`

## 📊 База данных

### Таблица `leads` (Заявки)

```typescript
{
  id: number;                    // Уникальный ID
  name: string;                  // Имя клиента
  phone: string;                 // Телефон
  email: string | null;          // Email
  message: string | null;        // Сообщение
  length: number | null;         // Длина (мм)
  width: number | null;          // Ширина (мм)
  height: number | null;         // Высота (мм)
  quantity: number | null;       // Тираж (шт)
  cardboardGrade: string | null; // Марка картона
  status: string;                // Статус: new | processing | shipped | completed
  response: string | null;       // Ответ администратора
  quizResults: string | null;    // Результаты квиза (JSON)
  fefcoCode: string | null;      // Код FEFCO
  createdAt: Date;               // Дата создания
  updatedAt: Date;               // Дата обновления
}
```

### Индексы
- `phone_idx` - для быстрого поиска по телефону
- `email_idx` - для быстрого поиска по email

## 🔒 Безопасность

### Реализованные меры

1. **Валидация данных**
   - Zod схемы для всех форм
   - Проверка на стороне сервера
   - Защита от SQL-инъекций (Drizzle ORM)

2. **Аутентификация**
   - HttpOnly cookies для сессий
   - Session storage для клиентской части
   - Автоматическое истечение сессий

3. **Защита роутов**
   - Проверка прав доступа в Server Actions
   - Редирект неавторизованных пользователей
   - Защита админ-панели секретным кодом

4. **CORS и CSRF**
   - SameSite cookies
   - Secure cookies в production

## 🐛 Известные проблемы

### Возможные проблемы и решения

1. **Ошибка подключения к БД**
   - Проверьте `DATABASE_URL`
   - Убедитесь, что PostgreSQL запущен
   - Проверьте права доступа

2. **Telegram не отправляет уведомления**
   - Проверьте `TELEGRAM_BOT_TOKEN` и `TELEGRAM_CHAT_ID`
   - Убедитесь, что бот добавлен в чат

3. **Админ-панель не открывается**
   - Проверьте `ADMIN_SECRET` в `.env`
   - Очистите кэш браузера
   - Проверьте cookies
